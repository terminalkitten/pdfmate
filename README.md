# PDFMate

Async / sync wrapper for Pyppeteer

### Install

    pip install pdfmate

### Setup for development

    poetry install -v --no-root

### Enable git-hooks with lint-staged

    npx mrm lint-staged
    npx husky install

### Run tests

    poetry run pytest tests/
